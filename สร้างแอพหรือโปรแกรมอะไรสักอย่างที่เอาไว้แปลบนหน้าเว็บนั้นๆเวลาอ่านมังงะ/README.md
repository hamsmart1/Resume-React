# 📖 Manga Web Translator (Chrome Extension)

ส่วนขยายเว็บบราวเซอร์ Google Chrome (Manifest V3) สำหรับครอบตัดและแปลภาษาบนหน้าเว็บเวลาอ่านมังงะ (Manga / Webtoon) เป็นภาษาไทยได้ทันทีแบบ Real-time โดยรองรับทั้งภาษาญี่ปุ่น, อังกฤษ, เกาหลี และจีน

## 🚀 ฟีเจอร์เด่น (Key Features)

* **Drag-to-Translate**: คลิกเปิดใช้งานแล้วลากครอบตัดกล่องคำพูด (Speech Bubble) เพื่อแปลภาษาได้ทันที
* **Draggable Speech Bubble**: กล่องข้อความแปลแสดงทับตำแหน่งเดิม และสามารถลากย้ายตำแหน่งได้เพื่อไม่ให้บังส่วนสำคัญของภาพ
* **Hybrid Engine**:
  * **Free Mode (Local OCR + Free Translate)**: ทำงานในเครื่อง 100% โดยใช้ `Tesseract.js` ค้นหาข้อความในตัวส่วนขยาย ร่วมกับระบบแปลภาษาฟรีของ Google Translate (ใช้งานได้ทันทีโดยไม่ต้องตั้งค่าคีย์ใดๆ)
  * **Advanced AI Mode (Gemini 1.5 Flash)**: ส่งรูปภาพที่ครอบตัดไปให้ Gemini วิเคราะห์และแปลความหมายในขั้นตอนเดียว มีความแม่นยำสูงมาก เข้าใจบริบท และให้ภาษาสวยงามเป็นธรรมชาติ
  * **Google Vision Mode**: ส่งภาพไปทำ OCR ด้วย Google Cloud Vision API สำหรับกรณีภาพตัวอักษรแนวตั้งหรือฟอนต์มังงะที่ซับซ้อน
* **Manifest V3 Compliant**: รันโค้ดและไลบรารีทั้งหมดแบบ Local ภายใน Extension ตัวเครื่อง ไม่มีการเรียกสคริปต์ภายนอก ปลอดภัยตามนโยบายรักษาความปลอดภัยของ Chrome
* **Multi-Language Support**: รองรับภาษาต้นทาง ญี่ปุ่น (ja), อังกฤษ (en), เกาหลี (ko) และจีน (zh) แปลเป็นภาษาไทย (th)

## 🛠️ วิธีการติดตั้ง (Installation Steps)

1. ดาวน์โหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ไว้ในเครื่องคอมพิวเตอร์ของคุณ
2. เปิดเบราว์เซอร์ **Google Chrome** แล้วเข้าไปที่หน้าตั้งค่าส่วนขยาย:
   * พิมพ์ `chrome://extensions` ในช่องที่อยู่เว็บ (URL bar) หรือกดปุ่มเมนู 3 จุดด้านขวาบน -> **ส่วนขยาย (Extensions)** -> **จัดการส่วนขยาย (Manage Extensions)**
3. เปิดใช้งาน **โหมดนักพัฒนาซอฟต์แวร์ (Developer mode)** ที่มุมขวาบนของหน้าส่วนขยาย
4. คลิกปุ่ม **โหลดส่วนขยายที่คลายการบีบอัดแล้ว (Load unpacked)** ที่มุมซ้ายบน
5. เลือกโฟลเดอร์โครงการนี้: `สร้างแอพหรือโปรแกรมอะไรสักอย่างที่เอาไว้แปลบนหน้าเว็บนั้นๆเวลาอ่านมังงะ`
6. ส่วนขยาย "Manga Web Translator" จะเปิดขึ้นและพร้อมใช้งานในแถบเครื่องมือของ Chrome!

## 📖 วิธีการใช้งาน (Usage)

1. ตรึง (Pin) ไอคอนส่วนขยายไว้บนแถบเครื่องมือ Chrome เพื่อความสะดวกในการเรียกใช้
2. ไปยังหน้าเว็บมังงะหรือภาพที่คุณต้องการอ่าน
3. คลิกที่ไอคอนส่วนขยายเพื่อเปิดเมนูตั้งค่า:
   * เลือกภาษาต้นทาง (เช่น ญี่ปุ่น หรือ อังกฤษ)
   * เลือก Engine การแปล (เริ่มต้นแบบ Free หรืออัปเกรดเป็น Gemini AI/Google Vision โดยการระบุ API Key)
4. คลิกปุ่ม **Select Area / เริ่มครอบตัด**
5. หน้าจอจะหม่นลงเล็กน้อย ลากครอบตัดข้อความหรือช่องคำพูดมังงะที่คุณต้องการแปล
6. รอประมวลผลสักครู่ กล่องข้อความแปลภาษาไทยจะแสดงทับตำแหน่งที่คุณเลือกทันที!
   * คุณสามารถกดกากบาท `×` มุมบนของกล่องเพื่อปิด หรือลากกล่องย้ายไปมาเพื่ออ่านภาพด้านหลังได้
